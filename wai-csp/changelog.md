# Changelog

## 1.0.0 (Initial Release)

- Initial release of wai-csp middleware package
- Type-safe Content Security Policy API for WAI applications
- Built-in nonce support with automatic generation and vault storage
- Predefined secure policies (defaultCSP, strictCSP)
- Helper functions for all CSP source types
- Framework-agnostic WAI middleware
- Comprehensive documentation and examples
